# Rock, Paper, Scissors Game ✂️🪨📄

A fun and interactive **Rock, Paper, Scissors Game** built with **HTML, CSS, and JavaScript**.  
Play against the CPU, choose your weapon, and see who wins! Perfect for learning JavaScript and having fun online.

## How it works
1. Choose one of the three weapons: Paper, Rock, or Scissors.
2. The CPU randomly selects a weapon.
3. If your choice beats the CPU's choice, you earn a point.
4. First to 10 rounds, the winner is announced.

## Features
- Interactive gameplay
- Score tracking
- Works on desktop and mobile
- Fun and educational
